
# gen_rfl

gen_rfl 是一个 C++ 反射代码生成工具，通过运行它可为 C++ 类自动生成反射函数。用户借助这些函数，能够获取结构体的成员变量并进行读取与修改。

与 GitHub 上其他反射库不同，gen_rfl 采用源码生成方式，针对每个 C++ 结构体生成可读、可查的 C++ 代码，兼具透明性与易调试性。
